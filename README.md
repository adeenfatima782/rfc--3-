# 🍔 RFC - Full-Stack Food Delivery & Restaurant Management System

RFC is a complete MERN stack web application designed to streamline online food ordering for customers and order management for restaurant administrators.

## 🚀 Features

### 👤 Customer Side:
- **Interactive Menu:** Browse food items across multiple categories (Burgers, Drinks, Sides) with instant search capabilities.
- **Smart Cart & Checkout:** Real-time calculation of Item Cost, Delivery Fees, and GST.
- **Discount/Coupon System:** Support for promotional codes (like `WELCOME50` for 50% discount).
- **Order History & Tracking:** Users can track their live order statuses (e.g., Pending, Accepted) from their profile dashboard.

### 👑 Admin Side:
- **Admin Dashboard:** Access-controlled panel strictly for authorized admin accounts to manage the restaurant backend.
- **Order Control Panel:** View all customer orders globally and change their status live.
- **Database Management:** Add, update, or remove products and discount coupons directly.

## 🛠️ Tech Stack

- **Frontend:** React.js, React Router
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (with Mongoose ODM)
- **Authentication:** JWT & Password Hashing
